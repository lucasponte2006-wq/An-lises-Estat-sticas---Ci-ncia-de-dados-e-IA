# 📊 Análises Estatísticas — Ciência de Dados e IA

Repositório com projetos de análise de dados desenvolvidos ao longo da graduação em Ciência de Dados. O objetivo é demonstrar habilidades práticas em estatística, manipulação de dados e visualização, utilizando diferentes linguagens e ferramentas.

---

## 🗂️ Projetos

### 📁 Análise Estatística de Funcionários de uma Empresa
**Linguagem:** R  
Análise exploratória de dados de funcionários de uma empresa, investigando variáveis como desempenho, perfil e distribuição de dados internos.

---

### 📁 Análise de Média Salarial com Base em Anos de Formado e Quantidade de Filhos
**Linguagem:** R  
Estudo estatístico da relação entre tempo de formação, quantidade de filhos e média salarial, aplicando técnicas de análise descritiva e inferencial.

---

### 📁 Análise de Variáveis Qualitativas e Quantitativas com Base nas Notas do ENEM por Região do País
**Linguagem:** Python  
Análise das notas do ENEM por região do Brasil, explorando variáveis qualitativas e quantitativas para identificar padrões e desigualdades regionais.

---

## 🛠️ Tecnologias Utilizadas

- Python
- R
- Google Colab
- Jupyter Notebook

> Novas linguagens e ferramentas serão adicionadas conforme o avanço do curso.

---

## ▶️ Como executar

Os notebooks podem ser abertos diretamente no Google Colab clicando no arquivo `.ipynb` e selecionando **"Abrir no Colab"**, ou executados localmente via Jupyter Notebook:

```bash
pip install jupyter
jupyter notebook
```
